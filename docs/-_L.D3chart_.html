Class: 'L.D3chart'

'L.D3chart'

new 'L.D3chart'(center, options)

L.D3chart is used to add dynamic charts on a leaflet map. It is specially useful to represent multiple data values associated to a given geographical coordinates.
Parameters:
Name Type Description
center L.Point
options D3chartOptions Object containing options to construct a chart.

Type Definitions

D3chartOptions

Options used to initialize/update a D3chart object.
Type:
  • object
Properties:
Name Type Attributes Default Description
type string <optional>
 "bar" Type of chart to create. Possible values are "bar" for barcharts, "pie" for pie charts, "polar-radius" and "polar-area" for polar area charts where values are represented either by the radius or the area of the slices.
data Array.<number> <optional>
 [1] Data values the chart has to represent.
maxValues Array.<number> <optional>
 [1] maximal absolute value the data could take. It can be a single numeric value or an array of values with same length as data. In the first case, all values will be represented with the same scale while in the second case, each value will have its own scale. This is useful when one wants to represent multiple variables that are not comparable.
fillColor string <optional>
 #4281e5 Color used to fill the shape when data contains only one value.
colorPalette Array.<string> <optional>
 d3.schemeCategory10 Array of colors used to fill each part of the chart representing one value of data.
width number <optional>
 60 Width of the chart when `type` equals 'bar' or maximal diameter of the chart for all other types.
height number <optional>
 60 Maximal height of barcharts.
opacity number <optional>
 1 Opacity of the chart.
showLabels boolean <optional>
 false Should values be displayed in the charts?
labelStyle string <optional>
 "font-family:sans-serif" CSS style to apply to labels
labelMinSize number <optional>
 8 Labels are automatically hidden if the label height is less than this number.
labelMaxSize number <optional>
 24 Maximal height of labels.
labelPadding number <optional>
 2 Padding to apply to labels.
labelPrecision number <optional>
 0 Number of significant digits to keep in labels. If it is equal to 0, values are displayed as is.
labelColor string <optional>
 "auto" Color to apply to labels. If "auto", text will be black or white depending on the background color.
labelText string | null <optional>
 null Used only when data has length one. Text to display inside the shape.
transitionTime number <optional>
 750 Duration in millisecondq of transitions.
